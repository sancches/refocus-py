# refocus-py
## Short projects to get back to programming in python.

### Project #1: unit converter
  - I made an unit converter that supports length, weight, volume and also temperature conversion.
  - Used dictionaries for storing linear conversion factors(length, weight and volume). 
  - For temperature conversion the process has to be different since it also uses offsets.
  - Implemented error testing for all inputs.

### Project #2: refactoring unit converter into a GUI program
  - Took the logic from the last project and made it into a visual program using tkinter.
  - Added a history to keep track of the past calculations, and a clear history button too
  - very simple looking, the focus was to make it functional and to grasp the concept of the library.
